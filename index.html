Ethical Company Investment Platform API

I took part in a semi-professional project, whereby a company contracted my post-graduate university to create the application over serveral trimesters with student engineers.


I developed an API which communicated with other ASX APIs to collect and structure company qualitative and financial statistics. According to ASIC regulation and other standards for guidance, my API fulfilled the data requirements to allow a user to make an informed decision on a share transaction.


A list of my contributions include:

  • An API that takes an ASX code, calls a chain of APIs, and then strutuctures and stores this data so that users can browse qualtitative information on ASX listed companies.

  • An API that periodically calls financial data APIs to collect and structure the most recent financial statistics.

  • A CRUD API so users can maintain a favourite company list.

  • An API that maintains a user's transaction history according to ASIC and other legislative guidelines.

  • Technical documentation for the aforementioned APIs.

Sqlite3 Script Generator

This web app allows users to design a relational database with an interactive table UI. Once the design is complete, users can generate a sqlite3 script which can be used to generate a sqlite3 database instance on a NodeJS server.


Socket.io supports realtime updates of the user's design, which saved to the mondoDB database. This in turn allows user's to sign-in and return to their design.

The next iterations of this application will focus on hosting the sqlite3 database, allowing users to populate it, and then perform data manipulations through a clean UI. The ultimate goal is to minimise the need for users to write sql script.
